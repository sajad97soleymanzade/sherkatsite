دستگاه جمع‌آوری داده (DAQ) مدل BD-100

دستگاه BD-100 یک سیستم جمع‌آوری داده چندکاناله است که برای ثبت و تحلیل سیگنال‌های سنسورها، از جمله لرزش، فشار، دما و جریان طراحی شده است. این دستگاه از دقت بالا، نرخ نمونه‌برداری قابل تنظیم، و ارتباط USB و Ethernet برای انتقال داده برخوردار است و در پروژه‌های تحقیقاتی و صنعتی کاربرد گسترده‌ای دارد.

  • پشتیبانی از چند نوع سنسور: لرزش، فشار، دما و جریان
  • نرخ نمونه‌برداری تا ۱۰۰kHz برای هر کانال
  • ارتباط دوگانه USB و Ethernet برای انتقال سریع داده
  • سازگاری با MATLAB، LabVIEW و Python API
  • طراحی فشرده با مصرف انرژی پایین
